LIMA

Linguagem de Marcação

É nesta máteria que somos introduzidos ao front-end, com as linguagens HTML e CSS, aprendendo a como programar com elas, fazendo layouts e nos aprofundando mais no CSS. Por esse motivo, vejamos dois vídeos que exemplificam alguns temas dados neste 1° semestre.

Como classificamos as linguagens do front-end?

O HTML é o que cria toda a estrutura do site, com todo o conteudo que estará presente, como videos, audios, imagens ou mesmo a parte escrita. O CSS é o que da estilo, forma e cor ao site, ele faz todo o design e da uma aparencia mais agradavel ao usuario. Já o JavaScript dá a ação e vida ao site, ou seja, ele faz a interação do usuario ao site ou aplicativo.

HARE

Hardware e Redes

Esta matéria é duas em uma, onde temos metade do semestre Redes, e a outra Hardware. No inicio dela, houve a introdução de alguns assuntos básicos para melhor compreensão e desenvolvimento da máteria, como o histórico do computador.

Rede

Em Redes aprendemos muitas coisas, como os bits e bytes, sobre os IPs, sendo o IPv4 e IPv6, suas clases e as máscaras de rede; sobre as conversões de base, com o binário, decimal, octal e hexadecimal; sobre topologia e os tipos de rede, como LAN e WAN; servidor; um pouco sobre as camadas OSI e outras coisas básicas de rede para um programador.

Hardware

Na parte de Hardware vemos conteúdos bases para montar um PC como as placas mães, as fontes, os processadores, os HDs, as placas de expansão, memória RAM, gabinete e entre outros, além de termos algumas aulas práticas no Senai, onde montamos um computador e fizemos a instalação do Windows 10, mexemos no boot e aprendemos mais sobre o setup.

FPOO

Fundamentos de Programação Orientada a Objeto

Nesta matéria aprendemos a programar em Java sendo ela orientada a objeto, ou seja, é um modelo de análise, projeto e programação de software baseado na composição e interação entre diversas unidades chamadas de 'objetos'. Nela, aprendemos alguns comando e seus respectivos funcionamentos como o 'if', 'while', 'for', entre outros, além de aprender sobre variáveis, classes, objetos e atribútos.

Sobre a linguagem: Java

Java é uma linguagem de programação e plataforma computacional. Existem muitas aplicações e sites que não funcionarão, a menos que você tenha o Java instalado, e mais desses são criados todos os dias. De laptops a datacenters, consoles de games a supercomputadores científicos, telefones celulares à Internet, o Java está em todos os lugares! Além disso, o Java é uma tecnologia usada para desenvolver aplicações que tornam a Web mais divertida e útil, e que não é igual ao JavaScript. O Java permite executar jogos, fazer upload de fotos, bater papo on-line, fazer tours virtuais e usar serviços, como treinamento on-line, transações bancárias on-line e mapas interativos. Se você não tiver o Java, muitas aplicações e websites simplesmente não funcionarão. Por default, o Java avisará a você automaticamente que novas atualizações estão prontas para serem instaladas. Para manter-se atualizado e proteger seu computador, é importante aceitar e instalar essas atualizações. Se você for avisado sobre uma atualização do Java no computador Windows e não lembrar de ter feito download dela e instalado, pode ser que o Java tenha vindo pré-carregado com seu novo computador.
